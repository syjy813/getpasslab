## 핵심 공식 / 정의

선반의 방호장치·부속장치.

| 장치 | 역할 |
|------|------|
| 칩 브레이커 | 연속 발생하는 칩을 짧게 끊는 공구 부착형 장치 |
| 덮개(실드) | 칩·절삭유 비산 방호 |
| 척 커버 | 척 회전부 접촉 방지 |
| 브레이크 | 급정지 |
| 방진구 | 지름에 비해 긴(길이가 지름의 12배 이상) 일감의 절삭 진동 방지 |

기계별 안전수칙 핵심.

| 기계 | 수칙 |
|------|------|
| 선반 | 장갑 착용 금지, 치수 측정·수리·청소는 정지 후, 칩은 브러시로 제거, 긴 일감은 방진구 |
| 밀링 | 커터 교환·측정은 정지 후, 칩은 회전 정지 후 브러시 제거, 커터에 얼굴 접근 금지, 장갑 금지 |
| 드릴 | 장갑 금지, 작은 일감은 바이스·클램프로 고정(손으로 잡지 않기), 구멍 관통 직전 이송 감속, 칩은 정지 후 제거 |

> ⚠️ **함정 주의**: 회전 기계 3종 공통 원칙 = 장갑 금지·정지 후 접촉·브러시 사용. "장갑을 끼고 작업한다", "회전 중 칩을 손으로 제거한다"류가 틀린 것 정답의 대부분. 절삭가공 분류 문제에서 프레스(소성가공)를 절삭가공에 끼우는 함정도 출제.

## 단위·기호 해설

| 용어 | 소속 | 주의점 |
|------|------|--------|
| 칩 브레이커 | 선반 (바이트 부착) | "방호장치의 일종인 칩 제거 기구" 표현으로 출제 |
| 방진구 | 선반 부속장치 | 진동 방지 — 방호장치와 구분 |
| 아버 | 밀링 커터 고정축 | 형삭기 구조부 함정과 교차 |

## 해석 / 의미

- 모든 수칙의 뿌리는 회전체 말림 방지 — 장갑·손 접촉·회전 중 조작 금지는 하나의 원리에서 파생
- 장치 명칭형(칩브레이커·방진구)과 수칙 소거형이 번갈아 나오므로, 장치는 "무엇을 막는가"와 함께 암기
- 신설 통합 챕터 — 선반·밀링·드릴 개별 기출이 많아 회차당 출제 확률이 가장 높은 영역 중 하나
