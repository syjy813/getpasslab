## 핵심 공식 / 정의

침투탐상검사(PT) — 표면 개구 결함에 침투력이 강한 적색 또는 형광성 침투액을 스며들게 한 뒤, 현상제로 빨아올려 결함의 위치·크기를 판별하는 비파괴검사.

작업 순서 6단계.

$$\text{전처리} \rightarrow \text{침투} \rightarrow \text{세척(제거)} \rightarrow \text{현상} \rightarrow \text{관찰} \rightarrow \text{후처리}$$

| 단계 | 내용 |
|------|------|
| 전처리 | 표면의 기름·녹·이물 제거 |
| 침투 | 침투액 도포, 개구 결함 내부로 침투 대기 |
| 세척(제거) | 표면에 남은 잉여 침투액 제거 |
| 현상 | 현상제 도포, 결함 속 침투액을 표면으로 흡출 |
| 관찰 | 직접 또는 자외선(형광)으로 결함 지시 확인 |
| 후처리 | 현상제 등 잔류물 제거 |

> ⚠️ **함정 주의**: 세척과 현상의 순서를 바꾼 나열이 대표 오답 — 잉여 침투액을 먼저 씻어내야 결함 지시만 남음. 침투탐상은 표면 "개구" 결함 전용, 내부 결함 검출 불가.

## 단위·기호 해설

| 용어 | 의미 | 주의점 |
|------|------|--------|
| 침투액 | 적색(가시) 또는 형광성 | 형광은 자외선등 관찰 |
| 현상제 | 침투액을 빨아올리는 백색 분말 | 흡출 원리 |
| 개구 결함 | 표면으로 열린 균열·기공 | 닫힌 내부 결함 불가 |

## 해석 / 의미

- 원리 = 모세관 현상 두 번 — 침투(결함 속으로), 현상(결함 밖으로). 이 왕복 구조를 이해하면 순서 암기가 논리로 대체됨
- 재질 제한이 거의 없음(금속·비금속) — 강자성체 전용인 자분탐상과의 대비 포인트
- 표면 결함 검사에서 육안 다음으로 간편해 현장 적용 빈도 높음 — 실무형 선지의 배경
