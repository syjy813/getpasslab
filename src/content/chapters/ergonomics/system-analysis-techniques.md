## 핵심 공식 / 정의

주요 시스템 안전 분석 기법.

| 기법 | 방향·성격 | 핵심 |
|------|-----------|------|
| PHA | 정성적 | 예비위험분석, 구상 단계 최초 적용. 식별 4범주: 파국적·중대(위기)·한계적·무시가능 |
| FMEA | 귀납적·정성적 | 부품 고장형태 → 영향 분석 |
| FTA | 연역적·정량적 | 톱사상에서 원인 전개 |
| ETA | 귀납적·정량적 | 초기 사상에서 결과 분기(사건나무) |
| HAZOP | 정성적 | 가이드워드로 설계 의도와의 이탈 탐색 |
| THERP | 정량적 | 인간 과오율(HEP) 예측 기법 |
| MORT | 관리 중심 | 원자력 등 고도 안전 확보 분야에서 관리·설계·생산·보전의 광범위한 안전 도모 |
| CA | 정량적 | 치명도 분석 — FMEA 보완 |

HAZOP 가이드워드.

| 워드 | 의미 |
|------|------|
| No/Not | 설계 의도의 완전한 부정 |
| More/Less | 양의 증가/감소 |
| As well as | 성질상의 증가(부가) |
| Part of | 성질상의 감소(일부만 수행) |
| Reverse | 설계 의도의 논리적 반대 |
| Other than | 완전한 대체 |

> ⚠️ **함정 주의**: 가이드워드 교차 매칭이 최다 함정 — "완전대체"는 Other than, "완전 부정"은 No/Not. PHA 4범주에 "통제적" 등 비범주를 끼운 선지, FTA를 귀납으로 서술한 선지가 반복 정답.

## 단위·기호 해설

| 축 | 구분 | 기법 |
|------|------|------|
| 방향 | 연역 vs 귀납 | FTA vs FMEA·ETA |
| 정량성 | 정량 vs 정성 | FTA·ETA·THERP vs PHA·HAZOP |
| 대상 | 설비 vs 인간 vs 관리 | HAZOP vs THERP vs MORT |

## 해석 / 의미

- 기법 선택의 논리 — 언제(수명주기 단계)·무엇을(설비/인간/관리)·얼마나 정밀하게(정성/정량) 볼 것인가의 3축 매트릭스
- 정의 문장의 키워드 1개로 판별 — "가이드워드=HAZOP, 인간 과오율=THERP, 원자력·광범위한 관리=MORT, 예비·최초=PHA"
- FMEA·FTA 개별 챕터의 상위 지도 역할 — 비교표를 먼저 잡으면 개별 기법의 특징 소거형이 전부 풀림
