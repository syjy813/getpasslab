## 핵심 정의

THERP(Technique for Human Error Rate Prediction)는 작업을 수행하는 사람의 실수 가능성을 정량적으로 추정하는 기법이다. 작업을 세부 단계로 나누고 각 단계의 성공·실패 확률과 작업 조건, 의존관계를 사용해 인간실수확률(HEP)을 계산한다.

THERP의 기본 흐름은 다음과 같다.

1. 분석할 작업과 성공 기준을 정의한다.
2. 작업을 관찰 가능한 인간 행동 단계로 분해한다.
3. 각 행동의 성공확률 또는 인간실수확률을 할당한다.
4. 단계 사이의 의존성, 작업 조건, 회복 가능성을 반영한다.
5. 조합된 결과로 전체 작업의 인간실수확률을 추정한다.

## 다른 분석 기법과의 구분

| 기법 | 중심 대상 | 접근 |
| --- | --- | --- |
| THERP | 인간 행동과 인간실수확률 | 작업 분해와 확률 정량화 |
| FTA | 시스템의 정상 상태 상실 또는 사고 톱 이벤트 | 톱다운 논리 조합 |
| ETA | 초기 사건 뒤의 가능한 결과 경로 | 사건 진행 분기 |
| FMEA | 부품·기능의 고장 형태와 영향 | 고장 형태별 검토 |
| MORT | 관리·조직 요인을 포함한 사고 원인 | 관리 체계 중심 분석 |

`20200822_031`은 THERP의 특징과 작업 단계의 조합을 묻는다. THERP는 시스템 분석 챕터에서 다루는 FTA·ETA와 함께 사용될 수 있지만, 인간 행동 확률을 직접 추정한다는 점이 구별점이다.

## 확률 모델의 주의점

작업 단계가 여러 개이면 각 단계의 확률을 단순히 더하거나 평균 내지 않는다. 문제에서 제시한 성공·실패 사건의 관계와 의존성을 먼저 확인하고, 해당 관계에 맞는 조합을 적용한다. 데이터가 없는 단계의 수치를 임의로 만들거나 최신 통계처럼 해석하지 않는다.

> **확정 주의**  THERP는 인간실수확률을 추정하는 기법이다. FMEA, FTA, ETA 또는 MORT의 이름을 THERP의 계산값으로 바꾸어 쓰지 않는다.

## 범위와 제외

이 챕터는 THERP의 목적, 작업 분해, 인간실수확률과 의존성의 정량화 개념을 다룬다. 특정 조직의 실제 오류율, 새로운 확률표, 최신 규정 또는 현장 위험성 평가 결과를 대신하지 않는다.

## 해석 / 함정

- `HEP`는 인간실수확률이며 작업 성공확률과 동일한 값이 아니다.
- 인간 행동 단계를 어떻게 나누는지가 결과에 영향을 준다.
- FTA·ETA와 함께 사용될 수 있어도 분석 대상과 논리 구조는 다르다.
- `20180428_026`처럼 인간실수확률 추정기법을 비교하는 문제에서는 THERP가 정량 추정 기법인지 확인한다.
