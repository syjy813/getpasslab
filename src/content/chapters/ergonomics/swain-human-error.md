## 핵심 분류

Swain의 인간실수 분류는 해야 할 작업을 빠뜨렸는지, 잘못 수행했는지, 순서·시간을 어겼는지, 불필요한 행동을 했는지로 오류를 구분한다.

| 유형 | 의미 |
| --- | --- |
| Omission error | 필요한 작업이나 절차를 수행하지 않음 |
| Commission error | 필요한 절차 대신 잘못된 행동을 수행함 |
| Sequential error | 필요한 작업의 순서를 잘못 수행함 |
| Time error | 작업 시점이나 타이밍이 잘못됨 |
| Extraneous act | 필요하지 않은 작업이나 절차를 추가로 수행함 |

`20190804_023`은 해야 할 단계의 생략을, `20210307_038`은 불필요한 작업을 수행한 경우를 묻는다. `20210814_021`은 상황을 오류 유형으로 분류하는 문제이며, `20200926_022`는 가스밸브를 잠그는 것을 잊은 사례를 생략 오류와 연결한다.

`20200822_022`처럼 각 영어 용어의 설명을 비교하는 문제에서는 omission·commission·sequential·extraneous의 정의와 시간 오류를 혼동하지 않는다.

## 다른 인간실수 이론과 구분

`20220424_022`의 실수·착오·위반 구분과 `20220305_040`의 James Reason 분류는 현재 챕터에 이미 연결된 인접 개념이다. 이 챕터는 이를 삭제하거나 재분류하지 않고, Swain의 작업 수행 오류 유형과 비교하는 데 사용한다. THERP는 인간실수확률을 정량화하는 기법이고 FMEA·FTA·ETA는 다른 분석 목적을 가지므로 이 챕터의 분류와 동일시하지 않는다.

> **확정 주의**  “작업을 하지 않음”은 omission, “불필요한 작업을 함”은 extraneous act다. 순서를 잘못 지킨 경우와 시점을 놓친 경우도 별도로 구분한다.

## 범위와 제외

이 챕터는 Swain의 omission, commission, sequential, time, extraneous 오류 분류와 사례 판단을 다룬다. THERP 계산, 일반 인간실수 원인 전체, FMEA·FTA·ETA, Rasmussen 등 다른 이론의 상세 설명은 범위에서 제외한다.

## 해석 / 함정

- 누락과 잘못된 수행은 각각 omission과 commission으로 구분한다.
- 수행 순서와 수행 시점을 바꾸어 읽지 않는다.
- 선택지에 다른 이론의 분류가 섞여 있으면 문제에서 요구하는 Swain 범위를 먼저 확인한다.
