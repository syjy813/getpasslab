## 계획의 목적

안전보건개선계획은 사업장의 안전과 보건 수준을 개선하기 위해 문제를 확인하고 개선 조치, 담당, 일정과 확인 방법을 정리하는 계획이다. 이 챕터는 저장소에서 정한 학습 범위를 정리하며, 외부 법령의 최신 서식이나 제출 요건을 새로 확정하지 않는다.

## 주요 항목

| 항목 | 학습 관점 |
| --- | --- |
| 시설·설비 개선 | 위험을 줄이기 위한 시설과 기계·설비의 개선 내용 |
| 안전보건관리체계 | 담당과 책임, 점검·보고 등 관리 구조 |
| 교육 | 개선 조치를 수행하기 위해 필요한 교육과 전달 방법 |
| 작업환경 개선 | 작업공간, 유해요인, 환경 조건의 개선 |
| 실행계획·일정 | 조치 순서, 담당, 기한과 완료 확인 |

항목을 나열하는 것에 그치지 않고, 확인된 문제와 개선 조치가 연결되는지 확인한다. 시설을 바꾸는 조치와 교육·관리체계를 보완하는 조치는 서로 대체 관계가 아니라 함께 계획될 수 있다.

## 범위와 제외

이 챕터는 안전보건개선계획의 목적과 시설·설비, 관리체계, 교육, 작업환경, 실행계획의 범주를 다룬다. 유해·위험방지계획서의 작성 자격·첨부서류, 안전관리규정 전체, 외부 법령에서 새 항목을 추정하는 작업은 범위에서 제외한다.

> **확정 주의**  계획서 항목은 저장소에서 승인된 학습 범위다. 실제 제출 서식과 현행 법정 항목은 사용 시점의 공식 기준을 별도로 확인해야 한다.

## 해석 / 함정

- 시설·설비 개선만 있고 실행 담당과 일정이 빠진 계획은 완결된 실행계획으로 보기 어렵다.
- 개선계획과 유해·위험방지계획서는 목적과 작성 맥락을 구분한다.
- 이 챕터에는 현재 직접 일치가 확정된 기출 관계가 없어 `questions` 배열을 비워 둔다.
