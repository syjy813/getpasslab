## 핵심 공식 / 정의

발파작업 안전대책 핵심.

| 항목 | 기준 |
|------|------|
| 장전구 | 마찰·충격·정전기에 의한 폭발 위험이 없는 것 |
| 발파공 충진재 | 점토·모래 등 발화성·인화성이 없는 재료 |
| 얼어붙은 다이너마이트 | 화기 직접 가열 금지 — 온수 등으로 융해 |
| 발파 전 | 도통시험 실시, 동력선과 이격 |
| 낙반 위험 조치 | 터널 지보공 설치, 록볼트 설치, 부석 제거 |

발파 허용진동치 (건물 기초 기준).

| 구조물 | 허용진동치 (cm/sec) |
|------|------|
| 문화재 | 0.2 |
| 주택·아파트 | 0.5 |
| 상가 | 1.0 |
| 철골콘크리트 빌딩·공장 | 1.0 ~ 4.0 |

터널 지보공 — 조립도(재료·치수 명시)에 따라 조립, 수시 점검사항 4종: 부재의 손상·변형·부식·변위·탈락, 부재의 긴압 정도, 접속부·교차부 상태, 기둥침하의 유무. 자동경보장치는 당일 작업시작 전 계기·검지부·경보장치의 이상 유무 점검. 암질 변화 구간·이상 암질 출현 시 암질판별(RQD·RMR·일축압축강도·탄성파속도 등) 실시.

> ⚠️ **함정 주의**: 철골콘크리트 빌딩 허용진동치를 0.8~1.0으로 낮춘 선지가 기출 정답(틀린 것) — 1.0~4.0이 맞음. 흙막이 지보공 점검(침하의 정도)과 터널 지보공 점검(기둥침하 유무)의 문구 차이도 교차 함정.

## 단위·기호 해설

| 용어 | 의미 | 주의점 |
|------|------|--------|
| 파일럿 터널 | 본 터널 전 지질·환기·배수 확인용 선행 터널 | 정의 단답 출제 |
| RQD·RMR | 암반 등급 지표 | 암질판별 기준 |
| 도통시험 | 발파 회로 연결 확인 | 전기발파 필수 절차 |

## 해석 / 의미

- 발파 대책의 공통 원리 = 의도하지 않은 점화원 차단(마찰·충격·정전기·화기)과 발파 후 확인(가스·불발) — 원리로 묶으면 소거형 대응 용이
- 허용진동치는 구조물 취약도 순서(문화재 < 주택 < 상가 < 철골) 그대로 — 서열만 기억해도 변조 선지 식별 가능
- 신설 통합 챕터 — 터널 지보공·경보장치·계측·굴착기계(파일럿 터널)까지 터널 계열 기출이 집중되는 최다 매칭 영역
