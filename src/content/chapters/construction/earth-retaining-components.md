## 핵심 공식 / 정의

흙막이 지보공의 주요 부재.

| 부재 | 역할 |
|------|------|
| 흙막이판 | 토압을 직접 받는 벽면재 |
| 말뚝(엄지말뚝) | 흙막이판을 지지하는 수직재 |
| 띠장 | 흙막이판·말뚝에 걸리는 토압을 수평으로 분산 — 흙막이판의 변형 방지 |
| 버팀대(스트럿) | 띠장을 반대편으로 지지하는 압축재 |
| 앵커 | 버팀대 대신 지반에 정착하는 인장재 |

조립도 명시사항 — 부재의 배치·치수·재질·설치방법과 순서.

정기 점검·보수 사항 (설치 후).

| # | 점검사항 |
|---|----------|
| 1 | 부재의 손상·변형·부식·변위·탈락의 유무와 상태 |
| 2 | 버팀대의 긴압(緊壓)의 정도 |
| 3 | 부재의 접속부·부착부·교차부의 상태 |
| 4 | 침하의 정도 |

> ⚠️ **함정 주의**: 부재가 아닌 것 유형에는 굴착기·터널 지보공 부재(록볼트 등)가 배치됨. 점검사항 오답으로 "지표수의 흐름 상태" 등 조문 밖 항목 삽입 — 4개 항목을 문구째 고정.

## 단위·기호 해설

| 용어 | 의미 | 주의점 |
|------|------|--------|
| 긴압 | 버팀대가 조여진 정도 | 점검 항목 2호 키워드 |
| 띠장 | 수평 연결재 | 비계의 띠장과 동명이물 |
| 히빙·보일링 | 굴착 저면 융기·분출 | 지보공 붕괴의 지반 측 원인 |

## 해석 / 의미

- 하중 흐름으로 부재를 이해 — 토압 → 흙막이판 → 말뚝·띠장 → 버팀대(또는 앵커) 순으로 전달되는 구조를 그리면 부재 식별이 자동화
- 점검 4항목은 "부재 자체 → 조임 → 연결부 → 지반"의 층위 — 구조로 외우면 이식 오답이 바로 보임
- 터널 지보공 점검사항(부재 손상·긴압·접속부·기둥침하)과 유사하므로 교차 출제 대비 필요
