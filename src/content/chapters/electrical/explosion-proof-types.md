## 핵심 공식 / 정의

| 구조 | 기호 | 원리 |
|------|------|------|
| 내압 방폭구조 | d | 내부 폭발압력에 견디고, 접합면·개구부 틈새로 화염이 외부에 전파되지 않게 함 |
| 압력 방폭구조 | p | 용기 내부에 보호가스(공기·질소)를 압입해 외부 폭발성 가스 침입 방지 |
| 유입 방폭구조 | o | 점화원이 될 부분을 기름 속에 넣어 가스와 격리 |
| 안전증 방폭구조 | e | 정상 운전 중 불꽃·과열이 생기지 않도록 안전도를 구조적으로 증강 |
| 본질안전 방폭구조 | ia, ib | 정상·고장 시 발생 불꽃 에너지가 착화 에너지 미만임을 시험으로 확인 |
| 비점화 방폭구조 | n | 정상 운전 중 점화원이 없음을 보증, 2종 장소 전용 |

내압(d) 구조의 필요충분조건 3가지 — ①내부 폭발압력에 견딜 것 ②접합면·개구부를 통해 외부 가스에 인화되지 않을 것 ③외함 표면온도로 주위 가스가 점화되지 않을 것.

> ⚠️ **함정 주의**: "종류가 아닌 것" 문제의 오답 제조법은 실존 구조의 명칭 변형(권선 방폭구조, 밀봉 방폭구조 등). 6가지 기호 d·p·o·e·ia/ib·n을 먼저 떠올리고 대조. 0종 장소에 사용 가능한 구조는 본질안전 ia뿐.

## 단위·기호 해설

| 기호 | 명칭 | 사용 가능 장소 | 주의점 |
|------|------|------|--------|
| d | 내압(耐壓) | 1·2종 | "압력에 견딤" — 압입하는 p와 혼동 금지 |
| p | 압력(압입) | 1·2종 | 보호가스로 내부 양압 유지 |
| o | 유입 | 1·2종 | 기름 = 격리 매체 |
| e | 안전증 | 1·2종 | 정상 상태만 보증 |
| ia / ib | 본질안전 | ia는 0종 가능 | ib는 1종까지 |
| n | 비점화 | 2종 전용 | 시험항목이 간소한 경제형 |

## 해석 / 의미

- 6가지 구조는 방폭 3대 개념(격리 d·p·o / 증강 e / 억제 ia·ib)의 구현형 — 원리에서 기호를 유도하면 소거형 문제는 자동 해결
- 내압 d가 출제 밀도 최상 — 정의(견딘다+전파 차단+표면온도 억제)와 시험항목(폭발강도·인화시험 등)까지 세트로 정리
- 장소 등급과의 연결(0종=ia, 2종=n)이 결합 출제의 핵심 고리
