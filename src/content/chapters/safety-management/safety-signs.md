## 표지의 4가지 유형

안전·보건표지는 전달하려는 행동과 위험의 성격에 따라 금지·경고·지시·안내로 구분한다.

| 유형 | 전달 목적 |
| --- | --- |
| 금지표지 | 특정 행동이나 출입을 하지 못하게 함 |
| 경고표지 | 위험물·위험장소·위험상태를 알림 |
| 지시표지 | 특정 보호구 착용 등 필요한 행동을 지시함 |
| 안내표지 | 비상구·피난소·통행 등 위치와 정보를 알림 |

`20210307_016`은 보안경 착용처럼 특정 행동을 요구하는 표지가 지시표지임을 확인한다. `20190303_017`과 `20210814_005`는 관계자 외 출입금지 표지의 적용 상황을 구분하는 문제다.

## 색상과 기본 모형

기출에서는 색채의 사용 사례와 기본 모형을 연결하거나, 경고표지 중 모형이 다른 항목을 찾게 한다. 대표적으로 빨강·노랑·파랑·녹색의 색채 의미와 표지 유형을 연결하되, 선택지에 제시된 색도 기준과 모형을 그대로 대조한다.

| 확인 항목 | 문제 풀이 기준 |
| --- | --- |
| 색채 | 특정 행위의 지시·사실 고지, 위험 경고, 정지·금지, 피난·안내 중 무엇을 나타내는지 확인 |
| 기본 모형 | 표지의 테두리·도형·내용이 문제에서 제시한 유형과 맞는지 확인 |
| 적용 상황 | 출입금지, 보호구 착용, 폭발성·방사성·부식성 물질 경고 등 구체적 문맥을 확인 |

`20220424_014`, `20210515_012`, `20180304_017`은 기본 모형을, `20210515_003`, `20200822_012`, `20200926_009`, `20180304_015`, `20180819_011`은 색채 또는 색도 기준을 묻는다. `20220305_014`와 `20180428_007`은 그림 또는 이미지 기반 문항이므로 `jpg 확필` 상태를 유지한다.

> **확정 주의**  색상 하나만 보고 표지 유형을 결정하지 않는다. 색채·기본 모형·표시 문구·적용 상황을 함께 확인한다.

## 범위와 제외

이 챕터는 금지·경고·지시·안내표지의 구분, 색상·형태·대표 적용 상황을 다룬다. 모든 표지 도안을 이미지 없이 복원하거나 국제표준과 현행 법령 전체를 확정하는 내용은 범위에서 제외한다.

## 해석 / 함정

- 안전모 착용·보안경 착용처럼 행동을 요구하는 표지는 지시표지인지 확인한다.
- 경고표지의 기본 모형이 같다고 단정하지 말고 문제의 도형을 확인한다.
- 이미지가 필요한 문항은 개념 관계와 공개 렌더링 가능 여부를 별도로 판단한다.
