## 핵심 공식 / 정의

강도율은 재해로 발생한 근로손실일수를 총근로시간으로 나누고 1,000을 곱한 값이다.

```text
강도율 = 근로손실일수 / 총근로시간 × 1,000
```

따라서 단위는 `1000근로시간당 근로손실일수`이며, 계산에서 총근로시간과 근로손실일수의 단위를 먼저 맞춘다. 강도율이 높다는 것은 같은 근로시간을 기준으로 손실된 근로일수가 많다는 뜻이다.

## 근로손실일수 환산

기출문제에서 사용하는 환산 기준은 다음과 같이 읽는다.

| 재해 구분 | 근로손실일수 |
| --- | ---: |
| 사망 또는 영구 전노동 불능 | 7,500일 |
| 영구 일부 노동 불능 | 등급별 기준일수 또는 `등급 기준 × 300 / 365` |
| 일시 전노동 불능 | 휴업일수에 `300 / 365`를 적용한 값 |

등급별 기준일수와 문제에서 제시한 환산 방식이 함께 주어지면 그 문제의 조건을 우선 적용한다. `20210515_015`처럼 도수율과 강도율이 함께 제시된 문제에서는 강도율의 손실일수 환산을 묻는지 확인한 뒤 계산한다.

## 계산 순서

1. 문제에서 정한 재해별 근로손실일수를 합산한다.
2. 총근로시간을 확인한다.
3. `근로손실일수 / 총근로시간 × 1,000`을 계산한다.
4. 선택지의 단위가 일수인지, 1,000근로시간당 일수인지 확인한다.

도수율은 재해 건수의 빈도를 나타내고 강도율은 손실일수의 크기를 나타낸다. 두 지표를 서로 바꾸어 대입하지 않는다. `20200822_019`와 `20180304_016`은 강도율 설명의 변환 규칙을, `20200926_007`, `20180428_004`, `20190427_012`는 제시된 강도율에서 손실일수를 역산하는 유형을 확인한다.

> **확정 주의**  강도율의 기준은 1,000근로시간이다. 총근로시간을 1,000으로 나누는 과정과 손실일수 환산을 누락하면 선택지가 달라진다.

## 단위·기호 해설

| 항목 | 의미 |
| --- | --- |
| 근로손실일수 | 재해로 일을 하지 못한 일수 또는 문제에서 정한 환산일수 |
| 총근로시간 | 강도율을 산출하는 분모 |
| 1,000 | 기준 근로시간 배율 |
| 강도율 | 1,000근로시간당 근로손실일수 |

## 범위와 제외

이 챕터는 강도율의 정의, 근로손실일수 환산, 강도율 계산과 도수율과의 구분을 다룬다. 재해 통계의 다른 지표 정의, 특정 법령의 최신 적용 여부, 개별 재해 원인 분석은 이 챕터의 범위가 아니다.

## 해석 / 함정

- `강도율 = 손실일수 / 총근로시간 × 1,000`의 순서를 유지한다.
- 강도율을 도수율처럼 재해 건수로 계산하지 않는다.
- 사망·장해·휴업의 손실일수 환산 조건을 문제 본문에서 다시 확인한다.
- 답을 고른 뒤에도 일수와 1,000근로시간당 일수의 단위를 확인한다.
