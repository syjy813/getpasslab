## 보호구 선택의 기본

보호구는 작업에서 예상되는 위험과 보호해야 할 신체 부위에 맞춰 선택한다. 보호구를 착용했다는 사실만으로 모든 위험이 제거되는 것은 아니며, 작업 전 위험을 확인하고 적합한 종류·등급·사용법을 함께 판단한다.

| 위험 또는 보호 부위 | 대표 보호구 |
| --- | --- |
| 머리 충격·낙하물 | 안전모 |
| 발의 충격·미끄럼 | 안전화 |
| 눈의 비산물·광선 | 보안경·차광용 보안경 |
| 분진 | 방진마스크 |
| 유기용제·유해가스 | 방독마스크와 해당 정화통 |
| 소음 | 방음 보호구 |
| 추락 | 안전대·추락방지대 |

## 유사 보호구 구분

방진마스크는 입자상 물질을, 방독마스크는 가스나 증기를 대상으로 한다. 방독마스크는 물질별 정화통과 표시 색을 확인해야 하며, `20220305_020`과 `20210515_007`은 이 연결을 선택지에서 구분하는 유형이다.

차광용 보안경은 자외선용·적외선용·복합용·용접용처럼 광선의 종류와 작업에 따라 구분한다. 방음 보호구의 기호, 안전모의 성능시험 항목, 방진마스크의 사용 조건도 보호 대상과 사용 환경을 먼저 확인한다.

`20220424_003`의 자율안전확인 보호구 표시 문제는 이미지 자산이 필요한 문항이므로 `review: "jpg 확필"`을 유지한다. 공개 페이지에서는 안전 필터가 이 문항을 계속 제외한다.

> **확정 주의**  산소결핍 장소를 방독마스크만으로 해결한다고 판단하지 않는다. 문제의 유해요인과 산소 조건을 먼저 확인한다.

## 범위와 제외

이 챕터는 안전모·안전화·보안경·방진·방독·방음 보호구의 기본 종류와 적용 판단을 다룬다. 보호구별 최신 인증 법령, 방음보호구 EP-1·EP-2·EM의 상세 규격, 개별 성능시험 전체는 범위에서 제외한다.

## 해석 / 함정

- 보호 대상 부위와 위험요인을 먼저 연결한다.
- 방진과 방독, 차광과 방음의 목적을 바꾸지 않는다.
- 보호구의 표시 색이나 등급은 문제에서 제시한 기준을 그대로 읽는다.
- 이미지나 빈 선택지가 있는 문항은 개념 관계와 공개 가능 여부를 별도로 판단한다.
