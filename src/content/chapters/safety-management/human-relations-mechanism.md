## 핵심 공식 / 정의

집단에서의 인간관계 메커니즘 5가지.

| 메커니즘 | 정의 |
|------|------|
| 동일화 | 다른 사람의 행동 양식이나 태도를 자기에게 투입하거나, 다른 사람 가운데서 자기와 비슷한 것을 발견하는 것 |
| 모방 | 남의 행동이나 판단을 표본으로 하여 그것과 같거나 가까운 행동·판단을 취하는 것 |
| 암시 | 다른 사람의 판단이나 행동을 논리적·사실적 근거 없이 받아들이는 것 |
| 투사 | 자기 속의 억압된 것을 다른 사람의 것으로 생각하는 것 |
| 커뮤니케이션 | 언어·몸짓·표정 등으로 의사·감정을 전달하는 것 |

> ⚠️ **함정 주의**: 동일화와 모방의 구분 — 동일화는 태도의 **내면화**("자기와 비슷한 것 발견"), 모방은 행동의 **표면 복제**. 암시의 키워드는 "근거 없이 받아들임". "메커니즘과 거리가 먼 것" 유형에는 사회행동 형태(경쟁·도피 등)가 배치됨.

## 단위·기호 해설

| 용어 | 키워드 | 주의점 |
|------|--------|--------|
| 동일화 | 투입·비슷한 것 발견 | 최다 출제 정의 |
| 암시 | 논리적 근거 없이 수용 | 유행·군중심리의 기제 |
| 투사 | 자기 것을 남에게 전가 | 방어기제와 공유 개념 |

## 해석 / 의미

- 5가지는 집단이 개인의 행동을 바꾸는 경로 — 안전 분위기(문화) 확산도 모방·암시·동일화의 메커니즘으로 설명됨
- 두문자 "동모암투커"로 목록을 고정하고, 정의는 각 메커니즘의 키워드 1개(발견·표본·근거 없이·전가·전달)로 압축
- 사회행동 기본형태(협대도융)와 목록 교차 함정이 상호 출제 — 두 챕터를 한 쌍으로 학습
